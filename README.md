# Contacts test application

[![Countries test app](https://img.youtube.com/vi/M9TLPkGriqY/0.jpg)](https://www.youtube.com/watch?v=M9TLPkGriqY)

### Stack:

- ElectronJs
- CryptoJs
- React
- Redux
- DevExtreme
- Styled Components
- TypeScript

### Run the app in the development mode:

`npm start`

### Build:

`npm run build`

### Make installer:

`npm run package`
